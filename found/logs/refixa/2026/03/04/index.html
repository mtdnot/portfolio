Agentira (RF-PRJ-005)

概要

  • AIエージェント教育ゲームプラットフォームのプロトタイプ開発
  • リポジトリ: https://github.com/CycleTree/Agentira
  • デプロイ先: https://cycletree.github.io/Agentira/

やったこと

技術選定

  • 言語/フレームワーク: Rust + Bevy 0.15
  • ビルドターゲット: WebAssembly (WASM)
  • 選定理由: AIエージェントがコード生成しやすい（全てコードベース、GUIエディタ不要）
  • Unity/Unreal を除外（GUI依存でAI操作困難）

プロトタイプ実装

  • 3Dシーン（床 + 動くエージェント + ライト）
  • エージェントの自律移動と境界反射
  • カメラ配置

CI/CD構築

  • GitHub Actions で WASM ビルド自動化
  • GitHub Pages へ自動デプロイ
  • trunk を使用した WASM ビルドパイプライン

解決した問題

  1. Rust バージョン問題: NixOS の Rust 1.82 が edition2024 未対応 → GitHub Actions で回避
  2. getrandom WASM対応: .cargo/config.toml で wasm_js backend 設定
  3. assets ディレクトリ問題: .gitkeep 追加で解決

構成ファイル

Agentira/
├── .cargo/config.toml  # WASM rustflags
├── .github/workflows/deploy.yml  # CI/CD
├── Cargo.toml          # Bevy 0.15 依存
├── Dockerfile          # ローカル開発用（未使用）
├── index.html          # trunk エントリポイント
└── src/main.rs         # Bevy ECS ゲームコード

苦労したこと

  • getrandom クレートの WASM 対応設定
  • Bevy の初回ビルド時間（CI で 11分）
